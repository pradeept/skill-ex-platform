# SkillBull
A skill exchange platform for [peer to peer learning](https://whatfix.com/blog/peer-to-peer-learning/).
</div>


## Run Locally

```shell
#clone the repo
git clone <repo-url>
cd <project-folder-name>

# run server
cd server 
npm install && npm run dev 

# run ui in another terminal
cd ../ui 
npm install && npm run dev 
````
